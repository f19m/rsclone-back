# rsclone-back

## stack
backend: express cors body-parser express-jwt express-json-validator-middleware jsonwebtoken argon2 
utils: dateformat
db: sequelize pg 
documentation: swagger-ui-express

## depoy
1. clone repositories
2. cd to project folder 
3. npm i
4. npm startDev
5. for test run: npm run test
